In order to run Pep8.1 on Linux, it is necessary to install the Qt libraries.

You can get the Qt libraries by typing:

sudo apt-get install libqtcore4

...or by downloading the Qt package from the Synaptics Package Manager.

We hope this isn't too much of an inconvenience!
